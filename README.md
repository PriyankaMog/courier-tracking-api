# 🚚 Courier Tracking API (Mini Logistics System)

A backend prototype for managing **shipments, tracking, and lifecycle workflows** in a logistics system.  
Built with **Node.js, Express.js, and MongoDB (Mongoose)**.

---

## ✨ Features
- 📦 **Shipment Creation** with sender & receiver details  
- 🔑 **Unique Tracking ID** generation  
- 🔄 **Shipment Lifecycle Management**  
  - created → picked-up → in-transit → out-for-delivery → delivered  
- 🕒 **Shipment History** with timestamps  
- 🔍 **Tracking System** (current status + full history)  
- 📑 **CRUD Operations** (Create, Read, Update, Delete)  
- 📝 **Request Logging Middleware** (method + route)  
- ✅ **API Testing** with Postman  

---

## 🛠️ Tech Stack
- **Backend:** Node.js, Express.js  
- **Database:** MongoDB + Mongoose  
- **Tools:** Postman (API testing)
